## 功能规格

### 1. 消息列表

**消息类型：**
- 文本消息
- 卡片消息（客户信息/业绩数据/任务列表）
- 快捷操作按钮

**消息展示：**
- 用户消息：右对齐，绿色气泡
- 机器人消息：左对齐，白色气泡
- 时间戳：显示在消息下方
- 头像：用户/机器人头像

**交互：**
- 下拉加载历史消息
- 点击卡片进入详情
- 长按消息复制

### 2. 快捷提示词

**默认提示词：**
1. "我想查看我的客户列表和跟进状态"
2. "帮我推进一下当前的服务事项和待办任务"
3. "定制我的机构经纪业务经营方案"

**交互：**
- 点击自动填入输入框并发送
- 支持横向滚动查看更多

### 3. 输入框

**功能：**
- 文本输入
- 语音输入（规划中）
- 附件发送（规划中）
- 表情选择（规划中）

**交互：**
- 输入时显示发送按钮
- 支持多行文本（自动增高）
- 点击发送或回车发送

### 4. 意图识别

**支持的意图：**
| 意图 | 示例 | 响应 |
|------|------|------|
| query_customer | "查一下XX公司" | 返回客户卡片 |
| query_tasks | "我的待办" | 返回任务列表 |
| query_performance | "本月业绩" | 返回业绩数据 |
| create_task | "帮我建个跟进任务" | 创建任务确认 |
| generate_plan | "生成经营方案" | 返回方案内容 |
| general_chat | 其他问题 | 通用对话回复 |

## 数据接口

| 接口 | 说明 |
|------|------|
| WS /ws/agent/chat | WebSocket对话 |
| POST /api/agent/chat | HTTP对话（备选） |
| GET /api/agent/history | 历史消息 |

---

## 异常处理

| 场景 | 处理 |
|------|------|
| 连接断开 | 自动重连，显示连接中提示 |
| 发送失败 | 显示重发按钮 |
| 无法理解 | "抱歉，我不太理解，您可以尝试..." |
| 服务异常 | "服务暂时不可用，请稍后重试" |

---

## 性能要求

- 首屏加载 < 1s
- 消息发送响应 < 500ms
- 支持1000+历史消息
