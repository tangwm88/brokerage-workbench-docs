## 功能规格

### 1. 业绩概览面板

**数据展示：**
- 月度目标完成率（环形进度图）
- 新增客户数（环比趋势）
- 签约客户数（环比趋势）
- 开户数（环比趋势）
- 收入（万元）

**交互：**
- 点击业绩指标进入详情页
- 环形图中心显示完成百分比

### 2. 待办事项

**数据展示：**
- 紧急待办数量（右上角badge）
- 待办列表（最多3条）
- 每条显示：标题、截止时间、AI建议

**交互：**
- 点击待办进入处理页面
- 滑动标记完成
- 点击"查看全部"进入待办列表

### 3. 快捷入口

**入口项：**
1. 客户开发 - 蓝色图标
2. 快速开户 - 绿色图标
3. 数字员工 - 橙色图标
4. 请示协同 - 红色图标

**交互：**
- 点击进入对应功能页面
- 长按可调整顺序（规划中）

### 4. 最近客户

**数据展示：**
- 最近更新的5个客户
- 每条显示：客户名称、状态、最后联系时间

**交互：**
- 点击客户进入详情页
- 点击"查看全部"进入客户列表

## 数据接口

| 接口 | 说明 |
|------|------|
| GET /api/dashboard/summary | 业绩摘要 |
| GET /api/dashboard/todos | 待办列表 |
| GET /api/dashboard/recent-customers | 最近客户 |

---

## 异常处理

| 场景 | 处理 |
|------|------|
| 数据加载失败 | 显示重试按钮 |
| 无待办事项 | 显示空状态插图 |
| 无最近客户 | 显示引导创建客户 |
