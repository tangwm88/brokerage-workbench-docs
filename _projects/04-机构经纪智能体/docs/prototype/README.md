## 📝 设计规范

### 色彩规范
- 主色：#1989fa（蓝色）
- 成功：#07c160（绿色）
- 警告：#ff976a（橙色）
- 危险：#ee0a24（红色）

### 字体规范
- 主字体：系统默认无衬线字体
- 标题：16-20px
- 正文：14px
- 辅助：12px

### 布局规范
- 移动端优先（375px基准）
- 安全区域适配
- 卡片圆角：12px
