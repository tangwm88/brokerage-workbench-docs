## 二、关键用户场景

### 场景1：新员工首次使用

```
登录 ──▶ 工作台首页 ──▶ 查看业绩看板
                          │
                          ▼
                    查看待办事项
                          │
                          ▼
                    点击"客户开发" ──▶ 录入第一个客户
```

### 场景2：跟进存量客户

```
工作台 ──▶ 查看"最近客户" ──▶ 点击客户
                                  │
                                  ▼
                            查看客户详情
                                  │
                                  ▼
                            点击"跟进" ──▶ 记录跟进结果
```

### 场景3：处理紧急事项

```
收到待办通知 ──▶ 打开工作台 ──▶ 查看紧急待办
                                        │
                                        ▼
                                  点击进入处理
                                        │
                                        ▼
                                  完成/转交/升级
```

## 三、机器人对话流程

### 意图识别流程

```
用户输入 ──▶ 关键词匹配 ──▶ 匹配成功？
                              │
                    ┌─────────┴─────────┐
                    │                   │
                    是                  否
                    │                   │
                    ▼                   ▼
            返回对应结果      调用大模型语义理解
                                        │
                                        ▼
                                  意图识别
                                        │
                                        ▼
                                  执行对应操作
```

### 多轮对话流程

```
用户：我想查看客户
        │
        ▼
机器人：请问您想查看哪个客户？可以输入客户名称或编号。
        │
        ▼

        │
        ▼

        1. XX投资管理有限公司
        2. XX投资有限公司
        请选择...
        │
        ▼
用户：1
        │
        ▼

```
