# 资产管理业务分析模型（5维度）

> **版本**：v1.0
> **日期**：2026-06-03
> **定位**：资产管理业务分析标准模型
> **目标角色**：资管产品官（主要调用者）
> **核心诉求**：这个产品能不能创设？能不能卖出去？卖出去后有没有风险？

---

## 维度1：产品需求（市场真实度）

**本质问题**：这个产品有没有真实市场需求？客户会不会买？竞争对手有没有类似产品？

**分析框架**：

```
【市场需求】
  ├─ 目标客户画像：年龄/资产规模/投资经验/风险偏好
  ├─ 需求来源：客户主动表达/市场调研发现/竞品启发/内部提案
  ├─ 需求紧迫度：急需（3个月内）/近期（3-6个月）/长期（6个月以上）
  └─ 需求验证：是否有客户意向确认/预认购/已签约意向书

【竞争格局】
  ├─ 竞品数量：同类型产品市场上已有多少只
  ├─ 竞品表现：同类产品近1年平均收益/最大回撤/规模变化
  ├─ 差异化空间：我方产品相比竞品的独特优势
  └─ 替代风险：客户不买这个产品，会买什么替代

【规模预判】
  ├─ 目标募集规模：XX亿元（基于客户需求调研）
  ├─ 市场容量：该类型产品近1年总发行规模/市场存量
  ├─ 我方份额预判：基于渠道能力/品牌认知度/历史表现
  └─ 规模合理性：目标规模/市场容量比例是否过高

【下一步动作】
  ├─ 需求真实 → 进入策略回测
  ├─ 需求不确定 → 扩大客户调研（增加50个样本）
  └─ 需求虚假 → 终止立项，记录原因
```

**输出格式**：
```
产品需求分析：[真实/不确定/虚假]

市场需求：
  ├─ 目标客户：XX（年龄XX-XX，资产规模XX-XX万）
  ├─ 需求来源：客户主动表达（已确认XX人意向）
  ├─ 需求紧迫度：近期（计划Q4发行）
  └─ 需求验证：有预认购意向书XX份

竞争格局：
  ├─ 竞品数量：市场已有XX只同类产品
  ├─ 竞品表现：近1年平均收益XX%，最大回撤XX%
  ├─ 差异化空间：我方产品XX优势（如：费率更低/策略更灵活）
  └─ 替代风险：客户可能购买XX产品（银行理财/信托）

规模预判：
  ├─ 目标募集：XX亿元
  ├─ 市场容量：该类型产品存量XX亿元
  ├─ 我方份额：预计占XX%（基于历史同类产品表现）
  └─ 合理性：目标/市场=XX% [合理]/[偏高]/[过高]

下一步：进入策略回测 / 扩大调研 / 终止立项
```

## 维度2：策略质量（投资可行性）

**本质问题**：投资策略能不能赚钱？风险多大？历史回测靠不靠谱？

**分析框架**：

```
【策略描述】
  ├─ 策略类型：股票多头/量化对冲/债券增强/多策略FOF
  ├─ 核心逻辑：基于什么规律/因子/信号做投资决策
  ├─ 投资范围：可投资资产类别（股票/债券/衍生品/商品）
  └─ 策略容量：策略最多能管多少钱（超过后收益下降）

【历史回测】
  ├─ 回测区间：XX年-XX年（至少5年，含牛熊周期）
  ├─ 回测年化收益：XX%（扣除费率后）
  ├─ 回测最大回撤：XX%（发生在XX-XX年）
  ├─ 回测夏普比率：XX（>1.0为优秀）
  ├─ 回测Calmar比率：XX（年化收益/最大回撤，>2.0为优秀）
  └─ 回测稳定性：月度胜率XX%，收益分布是否正偏

【风险测算】
  ├─ VaR（Value at Risk）：95%置信度下日/周/月最大损失XX%
  ├─ 情景分析：极端市场（如2008年/2015年/2020年）下的表现
  ├─ 压力测试：利率上行200bp/股市下跌30%/信用违约事件的影响
  ├─ 策略失效风险：什么情况下策略会失效（如：因子拥挤/市场结构变化）
  └─ 流动性风险：策略持仓的流动性如何（日均成交量/买卖价差）

【可行性判断】
  ├─ 策略回测通过（收益>基准+3%，最大回撤<20%）→ 进入结构设计
  ├─ 策略回测注意（收益>基准但回撤>20%）→ 优化风控参数
  └─ 策略回测不通过（收益<基准或回撤>30%）→ 终止或重新设计策略
```

**输出格式**：
```
策略质量分析：[通过/注意/不通过]

策略描述：
  ├─ 类型：量化对冲（市场中性）
  ├─ 核心逻辑：基于多因子模型（价值/质量/动量）构建股票组合，同时用股指期货对冲市场风险
  ├─ 投资范围：沪深A股（排除ST）+ 股指期货（IF/IC）
  └─ 策略容量：约50亿元（超过后冲击成本上升）

历史回测（2019-2024）：
  ├─ 年化收益：12.3%（扣除费率后）
  ├─ 最大回撤：8.5%（发生在2022年3-4月）
  ├─ 夏普比率：1.45
  ├─ Calmar比率：1.45
  └─ 月度胜率：68%（41/60个月正收益）

风险测算：
  ├─ VaR（95%）：周最大损失2.1%
  ├─ 情景分析：2008年金融危机期间模拟回撤12%（对冲后）
  ├─ 压力测试：利率上行200bp → 影响-1.2%；股市下跌30% → 对冲后影响-3%
  ├─ 策略失效风险：因子高度拥挤（如2021年初小盘因子崩盘）
  └─ 流动性风险：持仓日均成交量>5亿，买卖价差<0.1%

判断：通过（收益>基准+3%，回撤<20%，夏普>1.0）
下一步：进入结构设计
```

## 维度3：结构合规（监管可行性）

**本质问题**：产品结构是否合规？监管会不会批？备案能不能过？

**分析框架**：

```
【产品结构】
  ├─ 产品类型：公募基金/私募基金/券商资管计划/信托计划
  ├─ 产品结构：单层/双层/FOF/MOM
  ├─ 份额设计：A类（零售）/B类（机构）/C类（定制）
  ├─ 费率结构：管理费XX%/年 + 托管费XX%/年 + 业绩报酬XX%（高水位）
  └─ 开放机制：封闭式（XX年）/定期开放（每季度）/每日申赎

【合规要求】
  ├─ 产品类型合规：是否符合该类型产品的投资范围限制
  ├─ 杠杆比例：是否超过监管上限（如：券商资管杠杆≤200%）
  ├─ 集中度限制：单一资产/发行人占比是否超限（如：≤10%）
  ├─ 关联交易：是否涉及关联交易（需披露+审批）
  ├─ 投资者适当性：目标客户是否符合该产品的合格投资者要求
  └─ 备案要求：是否需要备案/注册/审批（如：公募基金需证监会注册）

【监管态度】
  ├─ 近期同类产品备案通过率：XX%（近3个月）
  ├─ 监管关注重点：当前监管对这类产品的态度（支持/中性/限制）
  ├─ 历史备案周期：平均备案时间XX天（从提交到获批）
  └─ 监管反馈常见问题：前5大备案反馈问题清单

【合规判断】
  ├─ 结构合规 → 进入备案准备
  ├─ 结构需调整 → 优化结构参数（如：降低杠杆/调整开放周期）
  └─ 结构不合规 → 终止立项，重新设计产品结构
```

**输出格式**：
```
结构合规分析：[合规/需调整/不合规]

产品结构：
  ├─ 类型：券商资管集合计划（大集合）
  ├─ 结构：单层（直接投资股票+债券）
  ├─ 份额：A类（零售，1000元起）+ B类（机构，100万起）
  ├─ 费率：管理费1.0%/年 + 托管费0.2%/年 + 业绩报酬20%（高水位）
  └─ 开放：封闭式2年，每季度开放申赎

合规要求：
  ├─ 投资范围：股票+债券+衍生品（符合券商资管投资范围）
  ├─ 杠杆比例：120%（≤200%，合规）
  ├─ 集中度：单一股票≤8%（≤10%，合规）
  ├─ 关联交易：无（不涉及关联方）
  ├─ 适当性：A类面向合格投资者（风险测评C4以上）
  └─ 备案要求：需向证监会备案（大集合新规要求）

监管态度：
  ├─ 近3月同类备案通过率：75%（12/16只通过）
  ├─ 监管态度：中性（无专项支持/限制政策）
  ├─ 平均备案周期：45天
  └─ 常见反馈：业绩报酬计提方式需明确（3/12只被要求补充）

判断：合规（所有硬性指标通过，备案周期可控）
下一步：进入备案准备
```

## 维度4：客户匹配（适当性匹配）

**本质问题**：这个产品适合哪些客户？客户的风险承受能力够不够？

**分析框架**：

```
【客户画像】
  ├─ 客户分层：高净值（>1000万）/ 富裕（100-1000万）/ 普通（<100万）
  ├─ 风险承受能力：C1（保守）/ C2（稳健）/ C3（平衡）/ C4（进取）/ C5（激进）
  ├─ 投资经验：新手（<1年）/ 有经验（1-5年）/ 资深（>5年）
  ├─ 投资目标：保值（年化3-5%）/ 增值（年化8-12%）/ 高收益（年化>15%）
  └─ 流动性需求：随时要赎回 / 1年内不动 / 3年以上不动

【产品风险等级】
  ├─ R1（低风险）：货币基金/国债
  ├─ R2（中低风险）：债券基金/银行理财
  ├─ R3（中等风险）：混合基金/FOF
  ├─ R4（中高风险）：股票基金/量化策略
  └─ R5（高风险）：衍生品/杠杆策略

【适当性匹配】
  ├─ 匹配规则：客户风险等级 ≥ 产品风险等级（如：C4客户可买R4及以下产品）
  ├─ 匹配结果：客户A（C4）→ 可买R1-R4产品 [匹配]
  ├─ 不匹配案例：客户B（C2）→ 不可买R4产品 [不匹配]
  └─ 匹配率：目标客户群中XX%符合适当性要求

【销售策略】
  ├─ 高匹配客户 → 重点推荐（>80%匹配率）
  ├─ 中匹配客户 → 培育后推荐（50-80%匹配率）
  └─ 低匹配客户 → 不主动推荐（<50%匹配率）
```

**输出格式**：
```
客户匹配分析：[匹配率高/中等/低]

客户画像：
  ├─ 目标客户：富裕阶层（100-1000万资产）
  ├─ 风险承受：C3（平衡型）为主，占比60%
  ├─ 投资经验：有经验（1-5年），占比55%
  ├─ 投资目标：增值（年化8-12%），占比70%
  └─ 流动性需求：1年以上不动，占比65%

产品风险等级：R3（中等风险）
  ├─ 策略类型：量化对冲（市场中性）
  ├─ 预期收益：年化8-12%
  ├─ 预期回撤：5-10%
  └─ 投资期限：建议持有2年以上

适当性匹配：
  ├─ 匹配规则：C3及以上客户可购买
  ├─ 目标客户匹配率：82%（C3及以上客户占目标客群82%）
  ├─ 高匹配客户：C4客户（30%）→ 可重点推荐
  └─ 需培育客户：C2客户（18%）→ 需先提升风险认知

销售策略：
  ├─ 高匹配客户（C3-C4，82%）→ 重点推荐
  ├─ 中匹配客户（C2，18%）→ 培育后推荐（推荐R2产品起步）
  └─ 不匹配客户（C1，<5%）→ 不推荐
```

## 维度5：投后风险（运行监控）

**本质问题**：产品运行后有没有风险？净值有没有异常？有没有关联交易？

**分析框架**：

```
【净值表现】
  ├─ 累计收益：成立以来累计收益XX%（年化XX%）
  ├─ 业绩基准：沪深300指数/中证500指数/中债总指数
  ├─ 超额收益：相对业绩基准的超额收益XX%
  ├─ 最大回撤：成立以来最大回撤XX%（发生在XX-XX）
  ├─ 回撤修复时间：从最大回撤恢复到前高用了XX天
  └─ 收益归因：收益来自选股（XX%）+ 择时（XX%）+ 行业配置（XX%）

【风险指标】
  ├─ 波动率：年化波动率XX%（与同类平均XX%对比）
  ├─ 下行风险：下行标准差XX%（仅计算亏损时的波动）
  ├─ 尾部风险：95%置信度下月度最大损失XX%
  ├─ 集中度风险：前10大持仓占比XX%（是否超过XX%警戒线）
  └─ 流动性风险：持仓日均成交量是否足够（如日均成交<1000万标记风险）

【关联交易监测】
  ├─ 关联交易清单：产品持仓中与我方关联方的交易
  ├─ 交易金额：关联交易金额占净值比例XX%
  ├─ 定价公允性：关联交易价格是否公允（与市场价对比）
  ├─ 披露完整性：关联交易是否已完整披露给投资者
  └─ 合规性：关联交易是否经过合规审批

【风险预警】
  ├─ 绿色（正常）：所有指标正常，无异常
  ├─ 黄色（注意）：某指标接近警戒线（如回撤接近15%）
  └─ 红色（预警）：某指标突破警戒线（如回撤>20%或关联交易异常）

【下一步动作】
  ├─ 绿色 → 常规监控（周报/月报）
  ├─ 黄色 → 加强监控（日报+风险排查）
  └─ 红色 → 立即预警 → 启动应急处置 → 通知客户/监管机构
```

**输出格式**：
```
投后风险分析：[绿色/黄色/红色]

净值表现（截至2026-06-03）：
  ├─ 累计收益：+15.2%（成立以来，年化8.7%）
  ├─ 业绩基准：中证500指数（同期+8.3%）
  ├─ 超额收益：+6.9%（相对基准）
  ├─ 最大回撤：6.8%（发生在2025-10至2026-01）
  ├─ 回撤修复：45天修复至前高
  └─ 收益归因：选股贡献+8.5% / 择时贡献-1.2% / 行业配置+0.6%

风险指标：
  ├─ 年化波动率：9.2%（同类平均12.5%，低于平均）
  ├─ 下行标准差：5.8%（仅亏损时波动）
  ├─ 尾部风险（95%）：月度最大损失-4.2%
  ├─ 集中度：前10大持仓占比35%（警戒线40%，正常）
  └─ 流动性：持仓日均成交>5亿，买卖价差<0.1%（正常）

关联交易监测：
  ├─ 关联交易：0笔（无关联方持仓）
  ├─ 关联交易金额：0%
  ├─ 定价公允性：无关联交易
  └─ 合规性：无关联交易需审批

风险预警：绿色（所有指标正常）
下一步：常规监控（周报更新净值/月报风险分析）
```

---

## 模型使用说明

1. **五个维度必须同时分析**，不能只挑一个
2. **每个维度分析后必须有下一步动作**
3. **禁止输出"整体评分"或"综合评级"**——只输出判断和动作
4. **分析结果必须在5个维度间自洽**（如：策略回测不通过的产品不能进入结构设计）
5. **维度3（结构合规）和维度4（客户匹配）的输出必须人工确认**——涉及合规风险
