# 禁用词语表（公司用语规范）

> 以下词语在公司智能体建设文档中**禁止使用**，需替换为对应推荐词。
> 更新时间：2026-05-30

| 禁用词 | 推荐词 | 说明 |
|--------|--------|------|
| **偏好** | **倾向 / 选择方向** | 讲具体倾向，不讲抽象偏好 |
| **流程** | **协同** | 讲工作协同方式，不讲线性流程 |
| **行业** | **产业** | 讲产业生态，不讲行业分类 |
| **场景** | **应用** | 讲应用落地，不讲场景概念 |
| **逻辑** | **规律** | 讲业务规律，不讲抽象逻辑 |
| **对齐** | **领会** | 讲领会意图，不讲对齐颗粒度 |
| **话术** | **沟通要点** | 讲沟通要点，不讲话术套路 |

---

## 补充说明

### 为什么要禁用这些词？
这些词多为互联网/AI行业黑话或抽象概念，在证券业务场景中：
- **不够具体**：业务人员听不懂"场景"是什么，但"应用"就明白是干什么用的
- **不够本质**："逻辑"太虚，"规律"是真实业务运行规则
- **不够真诚**："话术"暗示套路，"沟通要点"是真实需要传递的信息
- **不够准确**："行业"是简单分类，"产业"是生态链条
- **不够自然**："对齐"是互联网黑话，"领会"是真实认知行为

### 扩展原则
1. **能用动词不用名词**：如"赋能"→"让...能够"、"沉淀"→"积累"、"抓手"→"切入点"
2. **能用具体不用抽象**：如"颗粒度"→"细致程度"、"闭环"→"完整流程"、"打法"→"方法"
3. **能用白话不用术语**：如"心智"→"认知"、"链路"→"流程"、"私域"→"自有客户"
4. **不用英文缩写**：如"SOP"→"标准动作"、"KPI"→"考核指标"、"MVP"→"最简可行版"
5. **不用问题式维度**：如"靠不靠谱"→直接写本质名词（如"资金用途"）
