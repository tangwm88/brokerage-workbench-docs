# 投行债券发行业务分析模型（v2.0 具体可获取版）

> **版本**：v2.0  
> **日期**：2026-05-31  
> **核心调整**：维度从"概念描述"升级为"具体指标+数据来源+计算方式+获取路径"，确保每个维度信息真实可获取  
> **依据**：陶总5.28纪要 + 用户反馈（维度尽可能具体、信息可获取）

---

## 一、维度设计原则（v2.0升级）

| 设计原则 | v1.0（旧） | v2.0（新） |
|---------|-----------|-----------|
| **维度描述** | 概念性描述（如"现金流够不够"） | **具体指标**（如"经营性现金流净额/本期债券本息"） |
| **数据来源** | 笼统提及（如"财务数据"） | **明确来源**（如"Wind数据库-现金流量表/审计报告附注"） |
| **计算方式** | 定性判断 | **量化公式+阈值**（如"覆盖率=经营性现金流/本息，≥1.5倍通过"） |
| **获取路径** | 未说明 | **明确获取路径**（系统对接/手工录入/公开抓取） |

## 二、业务标准模型：5维度（具体指标版）

### 维度1：偿债能力（核心维度）

> **本质问题**：发行人到期能不能还本付息？  
> **核心指标**：用发行人**真实可动用的钱**覆盖**到期要还的钱**

#### 1.1 现金流覆盖能力（可直接从财报获取）

| 指标 | 计算公式 | 数据来源 | 获取路径 | 阈值标准 |
|------|---------|---------|---------|---------|
| **经营性现金流净额** | 现金流量表"经营活动产生的现金流量净额" | Wind/同花顺/iFinD-现金流量表 | 系统API直接拉取 | 近3年平均值 |
| **本期债券年本息** | 发行规模×票面利率+发行规模/期限（平均摊还法） | 拟发行方案参数 | 客户经理录入 | 按方案计算 |
| **现金流覆盖率** | 近3年平均经营性现金流净额 / 本期债券年本息 | 上述两项计算 | 系统自动计算 | **≥1.5倍[通过]** / 1.0-1.5倍[注意] / <1.0倍[不通过] |
| **自由现金流** | 经营性现金流净额-资本支出（购建固定资产等支付现金） | 现金流量表 | 系统API拉取 | 近3年是否为正 |
| **现金及等价物** | 货币资金+交易性金融资产（高流动性） | 资产负债表 | 系统API拉取 | 能否覆盖1年本息 |

#### 1.2 资产负债结构（可直接从财报获取）

| 指标 | 计算公式 | 数据来源 | 获取路径 | 阈值标准 |
|------|---------|---------|---------|---------|
| **资产负债率** | 总负债/总资产×100% | 资产负债表 | 系统API拉取 | **≤70%[通过]** / 70%-80%[注意] / >80%[不通过] |
| **有息债务/净资产** | (短期借款+长期借款+应付债券+长期应付款)/净资产×100% | 资产负债表 | 系统API拉取 | **≤40%[通过]** / 40%-60%[注意] / >60%[不通过] |
| **短期债务占比** | 短期有息债务/总有息债务×100% | 资产负债表 | 系统API拉取 | **≤50%[通过]** / >50%[注意] |
| **债券余额/净资产** | 已发行债券余额/净资产×100% | 已发行债券+资产负债表 | 系统API+中债登/上清所 | **公司债≤40%[通过]** / >40%[不通过] |
| **速动比率** | (流动资产-存货)/流动负债 | 资产负债表 | 系统API拉取 | **≥1.0[通过]** / <1.0[注意] |

#### 1.3 盈利能力（可直接从财报获取）

| 指标 | 计算公式 | 数据来源 | 获取路径 | 阈值标准 |
|------|---------|---------|---------|---------|
| **近3年净利润** | 利润表"归属于母公司股东的净利润" | 利润表 | 系统API拉取 | 是否连续为正 |
| **近3年净利润趋势** | (本年净利润-上年净利润)/上年净利润 | 利润表 | 系统自动计算 | 增长/波动/下滑 |
| **EBITDA** | 净利润+所得税+利息费用+折旧+摊销 | 利润表+现金流量表附注 | 系统API拉取 | 近3年平均值 |
| **利息覆盖倍数** | EBITDA / 利息费用 | 上述两项计算 | 系统自动计算 | **≥3.0倍[通过]** / 1.5-3.0倍[注意] / <1.5倍[不通过] |
| **净资产收益率(ROE)** | 净利润/平均净资产×100% | 利润表+资产负债表 | 系统API拉取 | 近3年平均是否>5% |

#### 1.4 同业对比（可获取，需手工整理或系统对接）

| 指标 | 计算方法 | 数据来源 | 获取路径 | 用途 |
|------|---------|---------|---------|------|
| **同业资产负债率均值** | 同行业上市公司资产负债率中位数 | Wind/同花顺-行业数据 | 系统API拉取或手工录入 | 判断发行人杠杆是否偏高 |
| **同业利息覆盖倍数均值** | 同行业上市公司利息覆盖倍数中位数 | Wind/同花顺-行业数据 | 系统API拉取或手工录入 | 判断偿债能力行业位置 |
| **同业现金流覆盖率均值** | 同行业经营性现金流/有息债务中位数 | Wind/同花顺-行业数据 | 系统API拉取或手工录入 | 判断现金流质量行业位置 |

### 维度2：发行方案

> **本质问题**：发多少、发多久、利率定多少、怎么还钱？  
> **核心指标**：方案参数是否在监管允许范围内、与市场可比、与发行人承受能力匹配

#### 2.1 规模参数（客户经理录入+系统校验）

| 指标 | 数值来源 | 获取路径 | 监管阈值 | 校验结果 |
|------|---------|---------|---------|---------|
| **拟发行规模** | 客户需求 | 客户经理录入 | - | - |
| **净资产规模** | 资产负债表"归属于母公司所有者权益合计" | 系统API拉取 | - | - |
| **累计债券余额/净资产** | 已发行债券余额（中债登/上清所查询）+ 本次拟发 / 净资产 | 系统对接中债登/上清所+本次录入 | **公司债≤40%** / 企业债≤60% | [通过]/[不通过] |
| **拟发行规模/净资产** | 本次拟发 / 净资产 | 本次录入+系统计算 | 需结合累计余额判断 | 是否超阈值 |
| **发行后资产负债率** | (当前总负债+本次拟发)/(当前总资产+本次拟发现金) | 系统计算 | 警戒80% | [通过]/[注意]/[不通过] |

#### 2.2 期限参数（客户经理录入+市场对比）

| 指标 | 数值来源 | 获取路径 | 判断标准 |
|------|---------|---------|---------|
| **拟发行期限** | 客户需求 | 客户经理录入 | - |
| **资金使用周期** | 募投项目计划 | 客户提供 | 期限应≥资金使用周期 |
| **期限匹配度** | 拟定期限 vs 资金使用周期 | 系统对比 | [匹配]/[不匹配] |
| **偿债高峰分析** | 未来各年到期债务（含本次） | 系统汇总已有债券+本次录入 | 是否与已有债务到期重叠 |
| **同业常见期限** | 同行业同评级近期发行债券期限中位数 | Wind/同花顺 | [匹配市场]/[偏长]/[偏短] |

#### 2.3 利率参数（市场数据+评级对应）

| 指标 | 数值来源 | 获取路径 | 判断标准 |
|------|---------|---------|---------|
| **发行人主体评级** | 中诚信/联合/大公/新世纪等评级报告 | 评级机构官网/Wind | AAA/AA+/AA/AA-/BBB+等 |
| **同评级同期限近期发行利率区间** | 近3个月同评级债券发行票面利率 | Wind-债券发行数据 | 系统自动统计min/max/中位数 |
| **建议利率区间** | 同评级区间±发行人资质调整 | 系统计算 | [有竞争力]/[中性]/[偏高] |
| **利率敏感性** | 利率上升50bp的年化成本增加 | 系统计算 | 增加XX万元/年 |
| **10年期国债收益率** | 中国债券信息网/Wind | 系统API拉取 | 当前基准 |
| **同评级信用利差** | 同评级债券发行利率-国债收益率 | 系统计算 | 近期变化：扩大/收窄/稳定 |

#### 2.4 还本付息方式（方案设计+现金流匹配）

| 指标 | 数值来源 | 获取路径 | 判断标准 |
|------|---------|---------|---------|
| **拟采用方式** | 客户需求/建议 | 客户经理录入 | 到期一次还本付息/分期还本/其他 |
| **还本节奏** | 按期限分摊 | 系统计算 | 各年还本金额分布 |
| **与现金流匹配度** | 还本节奏 vs 经营性现金流流入预测 | 系统对比 | [匹配]/[不匹配] |
| **偿债压力分布** | 未来各年还本付息合计/经营性现金流 | 系统计算 | [均匀]/[某年集中]/[压力小] |

#### 维度2输出格式

### 维度3：市场窗口

> **本质问题**：现在发债是不是好时候？投资者买不买？利率高不高？  
> **核心指标**：用市场真实交易数据判断时机

#### 3.1 利率环境（全市场公开数据，可自动获取）

| 指标 | 数据来源 | 获取路径 | 判断标准 |
|------|---------|---------|---------|
| **10年期国债收益率** | 中国债券信息网/Wind/中债估值 | 系统API拉取 | 当前值：XX% |
| **近3个月走势** | 每日10年期国债收益率 | 系统API拉取+自动计算 | 上升XXbp/下降XXbp/稳定 |
| **央行货币政策** | 央行官网-公开市场操作/MLF/LPR | 系统抓取央行公告 | 宽松/中性/收紧 |
| **市场资金面** | 银行间同业拆借利率(Shibor/DR007) | Wind/货币网 | 宽松/中性/紧张 |

#### 3.2 同类型债券发行情况（Wind公开数据，可自动获取）

| 指标 | 数据来源 | 获取路径 | 判断标准 |
|------|---------|---------|---------|
| **近1个月同评级债券发行只数** | Wind-债券发行数据 | 系统API拉取 | 发行活跃/一般/冷淡 |
| **近1个月同评级债券平均认购倍数** | Wind-债券发行数据 | 系统API拉取 | ≥2倍[旺盛]/1-2倍[一般]/<1倍[冷淡] |
| **近1个月同评级债券发行利率区间** | Wind-债券发行数据 | 系统API拉取 | 与历史对比：上升/下降/稳定 |
| **近期是否有同产业债券发行失败或取消** | Wind/市场新闻 | 系统API+舆情监控 | 有/无（如有，说明原因） |

#### 3.3 投资者需求（簿记建档历史数据+客户经理调研）

| 指标 | 数据来源 | 获取路径 | 判断标准 |
|------|---------|---------|---------|
| **我方投资者覆盖数量** | CRM系统/客户经理维护 | 系统查询 | 银行X家/基金X家/券商X家/保险X家 |
| **近期同类型债券我方承销认购倍数** | 内部簿记建档系统 | 系统查询历史数据 | 近3月平均XX倍 |
| **主要投资者当前配置需求** | 客户经理调研/投资者沟通 | 手工录入 | 旺盛/一般/谨慎 |
| **投资者结构预判** | 基于历史同类项目 | 系统参考历史 | 银行预计XX%/基金XX%/其他XX% |

#### 3.4 监管环境（公开政策文件，可系统抓取）

| 指标 | 数据来源 | 获取路径 | 判断标准 |
|------|---------|---------|---------|
| **近期债券审批通过率** | 交易所/发改委/银行间协会公示 | 系统抓取 | 近3月同类型通过率：XX% |
| **监管关注重点** | 交易所审核反馈函/政策文件 | 系统抓取 | 当前重点关注：XX |
| **是否有 supportive 政策** | 国务院/央行/证监会政策 | 系统抓取 | 绿色债券/科创债/乡村振兴等专项支持 |
| **发行人所属行业监管态度** | 近期同产业政策文件 | 系统抓取 | 支持/中性/限制（如城投/地产） |

#### 维度3输出格式
```
市场窗口分析

【利率环境】
├─ 10年期国债收益率：XX%（来源：中国债券信息网，日期：XXXX-XX-XX）
├─ 近3个月走势：下降XXbp（来源：系统计算，3月前XX%→现在XX%）
├─ 央行货币政策：宽松/中性/收紧（来源：央行官网-近期MLF/LPR操作）
└─ 市场资金面：宽松/中性/紧张（来源：Shibor 3M=XX%，DR007=XX%）
```

## 五、与定增模型的维度对比（具体指标版）

| 对比项 | 定增模型（核心指标） | 债券发行模型（核心指标） | 差异说明 |
|--------|---------------------|-------------------------|---------|
| **维度1** | 融资规模（拟募资金额/市值比例） | **偿债能力**（现金流覆盖率/资产负债率/利息覆盖倍数） | 债券必须量化偿债指标，定增不直接涉及 |
| **维度2** | 定价方案（折扣率/稀释比例） | **发行方案**（规模/期限/利率区间/还本付息方式） | 定增关心股本稀释，债券关心利率和偿还 |
| **维度3** | 市场时机（审核通过率/市场环境） | **市场窗口**（国债收益率/认购倍数/投资者需求/审批率） | 债券对利率环境更敏感，需关注流动性 |
| **维度4** | 资金用途（募投可行性/故事完整性） | **资金用途**（项目真实性/资金分配/效益测算/**偿债来源**） | 债券必须明确偿债来源，定增无此要求 |
| **维度5** | 合规风险（历史处罚/财务健康） | **增信与合规**（**信用评级**/增信措施/历史处罚/当前调查） | 债券有评级和增信体系，定增无 |
| **核心差异** | 关心"能不能发出去"（市场承接） | **关心"能不能还得起"（偿债能力）** | 债券有到期偿还义务 |
| **特有指标** | 股本稀释、锁定期、定增特有风险 | **现金流覆盖率、利息覆盖倍数、信用评级、增信覆盖倍数、偿债来源** | 债券特有量化偿债指标 |
