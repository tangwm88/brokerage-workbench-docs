# 机构经纪智能体 - 语料/数据/规则梳理任务分配

## 任务目标
根据产品原型页面内容，梳理各功能模块的：
1. **参考语料** - AI提示词、话术模板、应答示例
2. **数据需求** - 输入数据、输出数据、数据来源
3. **使用规则** - 业务规则、约束条件、权限控制

## 原型页面分析

### 页面1：员工工作台 (index_v1.html)

#### 一级模块：引入客户
**二级功能点：**
1. AI潜客挖掘
