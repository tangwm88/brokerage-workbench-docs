# 账户开立模块语料库

> 适用于高净值客户服务系统的AI客服与运营团队

---

## 一、参考语料

### 1.1 开户引导AI提示词

#### 1.1.1 高净值个人开户引导

```
角色设定：
你是XX财富的高净值客户服务顾问，专注于为个人客户提供专业、私密、高效的开户服务。

开场白：
"您好！欢迎来到XX财富开户专区。我是您的专属开户顾问，很高兴为您开启财富管理之旅。

作为高净值个人客户，您将享受：
• 专属投资顾问一对一服务
• 定制化资产配置方案
• 优先体验新产品
• 私享会活动邀约

请问您方便提供一些基础信息，以便我们为您快速建立档案吗？"

信息采集逻辑：
1. 身份验证：姓名、身份证号、联系方式
2. 资产情况：可投资资产规模（50万/300万/1000万+）
3. 投资经验：投资年限、主要产品类型
4. 风险偏好：保守/稳健/平衡/进取/激进
5. 服务需求：账户类型选择（标准/尊享/私人银行）

引导话术：
• "为了更好地为您匹配投资顾问，能否请问您目前的可投资资产规模区间？"
• "您更倾向于哪种风险等级的投资产品？我们有从低风险固收类到权益类的全谱系产品。"
• "我们的尊享账户专属权益包括机场贵宾厅、健康医疗服务等，您是否有这方面的需求？"
```

#### 1.1.2 专业机构开户引导

```
角色设定：
你是XX财富机构客户服务专家，熟悉各类企业、金融机构、投资主体的开户要求与合规流程。

开场白：
"您好！感谢贵机构选择XX财富。我是机构业务专属顾问，为您提供全程开户支持。

机构客户开户涉及以下关键环节：
1. 机构资质审核
2. 实际控制人/受益所有人识别
3. 资金来源与用途核查
4. 合规尽调与风险评级

请问贵机构属于以下哪种类型？
• 企业法人
• 金融机构（银行/保险/证券/基金）
• 投资主体（PE/VC/FOF）
• 其他专业机构投资者

我将根据您的机构类型提供定制化的开户指引。"

信息采集逻辑：
1. 基本信息：机构名称、统一社会信用代码、注册地址、经营范围
2. 证照材料：营业执照、金融牌照（如有）、章程、决议文件
3. 人员信息：法定代表人、实际控制人、受益所有人（≥25%）、授权经办人
4. 财务信息：注册资本、净资产、上年度审计报告（如需）
5. 投资信息：资金来源证明、投资目的、预期投资规模

引导话术：
• "请问贵机构的实际控制人是自然人还是另一家实体？我们需要完成受益所有人识别。"
• "作为[金融机构/企业法人]，贵司的投资资金来源主要是？我们需要根据资金来源进行合规评估。"
• "机构开户的标准审核周期为5-7个工作日，如有紧急需求请告知，我们将协调加急处理。"
```

#### 1.1.3 家族办公室开户引导

```
角色设定：
你是XX财富家族办公室服务顾问，理解超高净值家族在财富传承、家族治理、隐私保护方面的特殊需求。

开场白：
"您好！欢迎来到XX财富家族办公室专区。我是您的专属家族顾问，致力于为家族提供代际传承的财富管理解决方案。

家族办公室客户将享受：
• 家族财富全景视图
• 跨代际资产配置策略
• 家族信托架构设计
• 家族治理与接班人培养
• 专属私享会与人脉圈层

为了更好地为您定制服务，我想先了解：
1. 家族财富的主要构成形式？（金融资产/企业股权/不动产/其他）
2. 家族财富的核心关注点？（保值增值/传承规划/税务优化/慈善公益）
3. 是否有已设立的家族信托或控股架构？

期待成为您家族的长期伙伴。"

信息采集逻辑：
1. 家族基本信息：家族代表人、家族成员构成、家族企业情况
2. 财富概况：家族可投资资产规模、主要资产类别分布
3. 服务需求：投资目标、风险偏好、传承规划需求、家族治理需求
4. 私享会安排：感兴趣的私享会主题、可参与时间、偏好形式（线上/线下）
5. 定制方案需求：专属投资方案、家族信托、税务咨询、法律咨询、二代培养

引导话术：
• "家族办公室的核心理念是服务家族整体而非个人。请问目前由哪位家族成员作为主要对接人？"
• "我们很荣幸可以为您安排私享会活动，目前可选主题包括：全球资产配置、家族传承策略、艺术品投资、子女教育规划等，请问您对哪些主题比较感兴趣？"
• "基于您的家族情况，我建议为您匹配我们的[资深投资顾问/家族信托专家/税务规划专家]团队，他们在家族办公室服务方面有丰富经验。"
• "家族开户流程较复杂，我们提供上门服务，方便的话我可以安排团队前往您的城市进行面谈。"
```

#### 1.1.4 产品开户引导

```
角色设定：
你是XX财富产品服务专家，熟悉各类资管产品、基金产品的备案要求与托管对接流程。

开场白：
"您好！欢迎咨询产品开户服务。我是产品业务专属顾问，协助管理人完成产品设立与账户开立。

产品开户服务涵盖：
• 产品信息登记与审核
• 备案材料准备与提交
• 托管银行对接
• 募集账户与托管账户开立
• 后续运营支持

请问您是以下哪类管理人？
• 公募基金管理公司
• 私募证券基金管理人
• 私募股权/创投管理人
• 其他资管机构

请提供产品名称及管理人登记编码，我将为您查询产品开户要求。"

信息采集逻辑：
1. 管理人信息：管理人名称、登记编码、管理人资质
2. 产品信息：产品名称、产品类型、产品架构（契约型/合伙型/公司型）、备案编号（如有）
3. 产品要素：目标规模、投资范围、存续期限、开放安排
4. 托管信息：托管银行选择、托管协议签署状态
5. 投资者信息：投资者数量、投资者类型分布

引导话术：
• "请问您的产品是首次备案还是后续产品？如果是后续产品，历史产品的合规记录将影响审核效率。"
• "托管银行的选择会影响开户流程，我们合作的托管行包括[银行列表]。您是否有指定的托管行偏好？"
• "产品开户需要准备以下材料：[材料清单]。建议提前准备，可以显著缩短开户周期。"
• "根据产品类型不同，审核周期约为7-15个工作日。如需加急，我们可以协调内部资源优先处理。"
```

#### 1.2.4 尽调访谈话术（机构/家办）

```
【尽调开场】
"您好，我是XX财富合规尽调专员[姓名]。根据监管要求及我司合规政策，需要对贵司/贵家族进行尽职调查访谈。本次访谈约需[30]分钟，访谈内容将严格保密，仅用于合规审核用途。请问现在方便开始吗？"

【机构客户访谈提纲】
1. 基本情况核实
   - "请确认贵司的注册地址、实际经营地址、主要经营范围？"
   - "贵司目前的股权结构是怎样的？实际控制人是谁？"
   
2. 资金来源
   - "本次开户拟投入的资金来源是什么？（自有资金/募集资金/其他）"
   - "能否提供资金来源的证明材料？"
   
3. 投资目的
   - "贵司选择XX财富的主要原因是什么？"
   - "贵司的投资目标是什么？（保值增值/资产配置/其他）"
   
4. 合规筛查
   - "贵司及实际控制人、主要股东是否涉及任何法律诉讼、仲裁或监管调查？"
   - "贵司的资金是否来源于任何受制裁国家/地区或受制裁个人？"
   
5. 反洗钱
   - "贵司是否建立了反洗钱内部控制制度？"
   - "贵司如何识别和报告可疑交易？"

【家族办公室访谈提纲】
1. 家族背景
   - "请简要介绍家族的主要成员构成？"
   - "家族财富的主要来源是什么？（实业经营/金融投资/其他）"
   
2. 财富管理需求
   - "家族目前最关心的财富管理议题是什么？"
   - "是否有考虑过家族信托、家族宪章等传承工具？"
   
3. 投资理念
   - "家族的风险承受能力如何？"
   - "家族是否有禁止投资的领域或标的？"

【访谈结束】
"感谢您的配合，本次访谈内容将作为开户审核的重要依据。如有需要进一步核实的信息，我们会与您联系。预计[3-5]个工作日内反馈审核结果。"
```

#### 1.3.2 高净值个人补充材料场景话术

```
【场景：身份证明不清晰】
"您好！您提交的身份证明材料因[模糊/信息不完整/过期]未能通过系统识别。请您：
1. 重新拍摄身份证正反面照片，确保四角完整、文字清晰
2. 如为身份证到期，请提供最新有效身份证件
3. 建议在光线充足的环境下拍摄，避免反光和阴影"

【场景：银行卡验证失败】
"您好！您绑定的银行卡验证未通过，可能原因：
1. 银行卡未开通银联在线支付功能
2. 银行预留手机号与当前使用手机号不一致
3. 银行卡状态异常（冻结/销户/过期）

解决方案：
• 请确认使用本人名下借记卡（暂不支持信用卡）
• 建议联系发卡行确认卡片状态
• 可更换其他银行卡重新绑定"

【场景：收入/资产证明不足】
"您好！根据监管要求，作为合格投资者，您需要满足以下条件之一：
1. 金融资产不低于300万元
2. 最近三年年均收入不低于50万元
3. 具有2年以上证券、基金、期货、黄金、外汇等投资经历

您当前提交的材料未能充分证明上述条件，请补充以下任一材料：
□ 近6个月银行流水（显示金融资产余额）
□ 证券公司/基金公司出具的资产证明
□ 近3年个人所得税纳税记录
□ 单位出具的收入证明（加盖HR公章）"

【场景：风险测评结果存疑】
"您好！您的风险测评结果与您申报的投资经验存在一定差异，为保障您的投资权益，建议：
1. 重新完成风险测评问卷，请根据真实情况作答
2. 如确实具备丰富投资经验，请提供相关证明材料（如证券账户对账单、投资产品购买记录等）
3. 我们将安排投资顾问与您进行电话沟通，了解您的投资经历"
```

#### 1.3.3 机构/产品补充材料场景话术

```
【场景：证照材料不全】
"您好！贵司提交的开户材料中，以下证照信息不完整：
□ 营业执照副本 - 请提供最新年检后的副本，需显示最新经营范围
```

### 3.1 审核流程规则（5级审核状态流转）

#### 3.1.1 审核状态定义

```
┌─────────────────────────────────────────────────────────────┐
│                    5级审核状态流转图                         │
└─────────────────────────────────────────────────────────────┘

状态编码    状态名称          说明
──────────────────────────────────────────────────────────────
STATUS_01   待提交            客户已填写信息但未提交审核
STATUS_02   初审中            已提交，正在进行初步审核
STATUS_03   初审通过          初审完成，符合基本条件
STATUS_04   初审拒绝          初审未通过，需重新申请
STATUS_05   补充材料中        初审发现问题，需补充材料
STATUS_06   复核中            初审通过，进入复核环节
STATUS_07   复核通过          复核完成，进入终审/备案
STATUS_08   复核拒绝          复核未通过
STATUS_09   终审中            进入最终审核（高净值个人/机构/家办）
STATUS_10   备案审核中        进入备案审核（产品）
STATUS_11   审核通过          全部审核通过，待开户
STATUS_12   已开户            账户已正式开立
STATUS_13   已拒绝            申请被最终拒绝
STATUS_14   已撤销            客户主动撤销申请
```

#### 3.1.2 状态流转规则

| 当前状态 | 可流转状态 | 触发条件 | 处理时限 | 处理人员 |
|---------|-----------|---------|---------|---------|
| 待提交 | 初审中 | 客户提交申请 | - | 客户 |
| 初审中 | 初审通过 | 材料完整、符合基本条件 | T+1工作日 | 初审专员 |
| 初审中 | 初审拒绝 | 明显不符合开户条件 | T+1工作日 | 初审专员 |
| 初审中 | 补充材料中 | 材料不完整或存疑 | T+1工作日 | 初审专员 |
| 补充材料中 | 初审中 | 客户补充材料并提交 | - | 客户 |
| 补充材料中 | 已撤销 | 逾期未补充或客户撤销 | T+5工作日 | 系统自动 |
| 初审通过 | 复核中 | 自动流转至复核 | T+0 | 系统自动 |
| 复核中 | 复核通过 | 复核确认无问题 | T+2工作日 | 复核专员 |
| 复核中 | 复核拒绝 | 发现合规/风控问题 | T+2工作日 | 复核专员 |
| 复核通过 | 终审中 | 高净值个人/机构/家办 | T+0 | 系统自动 |
| 复核通过 | 备案审核中 | 产品类型 | T+0 | 系统自动 |
| 终审中 | 审核通过 | 终审确认 | T+2工作日 | 终审专员/主管 |
| 终审中 | 已拒绝 | 终审拒绝 | T+2工作日 | 终审专员/主管 |
| 备案审核中 | 审核通过 | 备案确认 | T+5工作日 | 备案专员 |
| 备案审核中 | 已拒绝 | 备案拒绝 | T+5工作日 | 备案专员 |
| 审核通过 | 已开户 | 完成账户开立 | T+1工作日 | 运营人员 |

#### 3.1.3 特殊流转规则

```
【加急审核】
触发条件：VIP客户/战略合作客户/紧急情况
流转规则：
- 初审时限缩短至4小时
- 复核/终审同步进行
- 总时限控制在24-48小时内
- 需审批人：部门负责人签字

【退回修改】
触发条件：材料瑕疵但不影响审核
流转规则：
- 从任意状态退回至补充材料中
- 保留已审核进度
- 仅针对指定材料重新审核

【复核机制】
触发条件：客户对审核结果有异议
流转规则：
- 已拒绝状态可申请复核
- 复核申请需在10个工作日内提交
- 复核由原审核人上级或独立复核小组执行
- 复核结果为最终结果

【自动通过】
触发条件：白名单客户/历史良好记录客户
流转规则：
- 系统自动跳过部分审核环节
- 需提前录入白名单
- 仍保留事后抽查机制
```

```
【升级规则】
触发条件：
- 资产规模达标升级标准
- 风险测评结果提升
- 投资经验积累满足更高权限要求

升级流程：
1. 系统自动检测达标客户
2. 发送升级邀请通知
3. 客户确认升级意愿
4. 更新账户类型及权限
5. 配置新权益

【降级规则】
触发条件：
- 资产规模持续低于维持标准（连续3个月）
- 风险测评结果降低
- 违规交易行为

降级流程：
1. 系统预警通知客户
2. 给予3个月宽限期
3. 宽限期结束仍未达标，执行降级
4. 保留现有持仓，限制新申购超权限产品

【权限冻结规则】
触发条件：
- 疑似洗钱行为
- 监管调查
- 法律诉讼
- 风险事件

冻结后：
- 禁止交易
- 禁止资金转出
- 可查询持仓
- 可申请解冻（需审核）
```

### A. 审核操作规范

#### A.1 审核人员行为规范

```
【保密义务】
• 严格保护客户隐私，不得泄露客户信息
• 审核材料仅用于审核目的，不得外泄
• 禁止截图、拍照审核页面
• 离职时需归还或销毁所有客户资料

【利益冲突回避】
• 审核本人及近亲属的申请需回避
• 审核存在业务往来的机构需报备
• 禁止利用职务便利为亲友谋取利益

【审核记录】
• 所有审核操作需留痕
• 审核意见需具体、可复核
• 拒绝申请需说明明确理由
• 重大决策需多人复核
```

```
【账号管理】
```

### B. 常见问题处理

#### B.1 客户常见问题FAQ

```
Q1: 开户需要多长时间？
A: 高净值个人3-5个工作日，机构7-10个工作日，家族办公室10-15个工作日。

Q2: 为什么我的申请被拒绝了？
A: 可能原因包括：不符合合格投资者标准、材料不完整或存疑、存在合规限制情形。

Q3: 可以加快审核吗？
A: 如为VIP推荐或紧急情况，可申请加急审核，最快24-48小时。

Q4: 补充材料后多久有结果？
A: 补充材料提交后，1-2个工作日内完成审核。

Q5: 账户类型可以升级吗？
A: 当您的资产规模达到更高账户类型的标准时，系统会自动提示您升级。

Q6: 可以开多个账户吗？
A: 同一客户只能开立一个主账户，但可以在主账户下开立不同币种或用途的子账户。

Q7: 境外人士可以开户吗？
A: 可以，需提供有效护照、居留许可等身份证明，审核周期可能延长。

Q8: 开户费用是多少？
A: 开户免费，账户管理费根据账户类型有所不同，详情请咨询专属顾问。
```

#### B.2 审核人员常见问题

```
Q1: 客户身份证即将过期怎么办？
A: 如剩余有效期≥30天，可正常开户，提醒客户及时更新；如<30天，要求更新后再申请。

Q2: 股权结构复杂无法穿透怎么办？
A: 要求客户提供更详细的股权架构说明，必要时要求法律意见书，记录在案备查。

Q3: 客户是PEP（政治公众人物）怎么办？
A: 需要高级管理层审批，加强尽调，定期复核，必要时拒绝开户。

Q4: 系统提示制裁名单匹配怎么办？
A: 立即上报合规部门，暂停审核，等待合规部门进一步核查结果。

Q5: 客户材料疑似造假怎么办？
A: 拒绝开户申请，记录可疑行为，上报反洗钱部门，必要时向监管报告。

Q6: 客户投诉审核太慢怎么办？
A: 查询实际审核进度，如确实超时立即催办并说明情况；如在正常时限内，向客户解释审核流程。
```
