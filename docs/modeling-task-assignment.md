# 机构经纪智能体 - 数据建模方案任务分配

## 任务概述
按现有二级功能点逐个梳理，整理出相应的数据建模方案。

## 功能模块划分

### 模块1：员工工作台 - 引入客户模块
**功能点：**
1. AI潜客挖掘 (showAIProspecting)
   - 线索录入
   - 客户评估
   - 线索分配

2. 找过来的客户 (showInboundClients)
   - 官网咨询
   - 转介绍
   - 活动获客

3. 主动出击获客 (showOutboundClients)
   - 陌拜记录
   - 渠道拓展
   - 转化跟踪

**建模需求：**
- 客户线索表
- 客户评估模型
- 跟进记录表
- 转化漏斗分析

---

## 子任务分配

创建10个子智能体，分别处理上述10个模块的数据建模方案。

每个子任务需要输出：
1. 实体关系图（ER图文字描述）
2. 数据表结构设计
3. 关键字段说明
4. 业务规则定义
